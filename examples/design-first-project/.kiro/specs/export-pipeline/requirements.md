# Requirements Document

## Introduction

Requirements distilled from the approved design (design-first workflow).

## Requirements

### Requirement 1

**User Story:** As a customer, I want to export my invoices, so that I can process them in my own tools.

#### Acceptance Criteria

1. WHEN a customer requests an export THEN the system SHALL deliver a download link within five minutes
2. WHEN an export fails permanently THEN the system SHALL notify the customer with a retry option
